## 🌟 ভিশন (The Vision)
**আমার হিসাব** শুধুমাত্র একটি খাতা নয়, এটি আপনার আর্থিক জীবনের ডিজিটাল সঙ্গী। আপনার প্রতিদিনের আয়-ব্যয়কে নিখুঁতভাবে ট্র্যাক করতে এবং ভবিষ্যতের জন্য সঠিক সিদ্ধান্ত নিতে এটি আপনাকে সাহায্য করবে। কোনো জটিলতা ছাড়াই আপনার পকেটের সব হিসাব এখন হাতের মুঠোয়।

## 🏗 ইনস্টলেশন এবং গাইড

### ১. ক্লোনিং (Getting Started)
প্রথমে রিপোজিটরি ক্লোন করুন এবং প্রজেক্ট ডিরেক্টরিতে প্রবেশ করুন:
```bash
git clone https://github.com/nullcove/ledger-knox.git
cd ledger-knox
```

### ২. এনভায়রনমেন্ট কনফিগারেশন
`.env.example` থেকে `.env.local` তৈরি করুন:
```bash
cp .env.example .env.local
```

### ৩. রান করা (Execution)
ডিপেন্ডেন্সি ইন্সটল করে ডেভেলপমেন্ট সার্ভার চালু করুন:
```bash
npm install && npm run dev
```
