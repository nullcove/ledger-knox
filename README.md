## 💎 কেন "আমার হিসাব" ব্যবহার করবেন?

আপনার কষ্টার্জিত টাকার হিসাব রাখা এখন আরও সহজ। **আমার হিসাব** প্রজেক্টটি ডিজাইন করা হয়েছে ব্যবহারকারীর সহজবোধ্যতা এবং আধুনিক প্রযুক্তির সমন্বয়ে। এটি শুধুমাত্র একটি অ্যাপ্লিকেশন নয়, এটি আপনার ডিজিটাল ক্যাশ-বুক।

### ⚡ প্রিমিয়াম ফিচারসমূহ:

- **📊 স্মার্ট ড্যাশবোর্ড:** এক নজরে আপনার মোট আয়, ব্যয় এবং বর্তমান ব্যালেন্স দেখার সুবিধা।
- **🎨 ইন্টারঅ্যাক্টিভ ডিজাইন:** Motion.js এবং Tailwind CSS দিয়ে তৈরি চোখ ধাঁধানো এনিমেশন এবং স্মুথ ট্রানজিশন।
- **🔐 সর্বোচ্চ নিরাপত্তা:** আধুনিক অথেনটিকেশন সিস্টেমের মাধ্যমে আপনার ডেটা রাখা হয় সম্পূর্ণ সুরক্ষিত।
- **📱 ফুল রেসপন্সিভ:** মোবাইল, ট্যাবলেট বা ডেস্কটপ - সব ডিভাইসেই পাওয়া যাবে সেরা অভিজ্ঞতা।
- **📄 রিপোর্ট জেনারেশন:** মাসিক এবং বাৎসরিক হিসাবের বিস্তারিত রিপোর্ট দেখার সুবিধা।

## ⚙️ সেটআপ গাইড (How to Install)

### ১. রিপোজিটরি ক্লোন করুন
```bash
git clone https://github.com/nullcove/ledger-knox.git
cd ledger-knox
```

### ২. ডিপেন্ডেন্সি ইন্সটল করুন
```bash
npm install
```

### ৩. কনফিগারেশন
`.env.example` ফাইলটি রিনেম করে `.env.local` করুন এবং আপনার ডাটাবেস ইউআরএল সেট করুন।

### ৪. প্রজেক্ট চালু করুন
```bash
npm run dev
```
