## 🛠 প্রযুক্তি (Tech Stack)

- **Frontend:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Components:** Radix UI / Shadcn UI
- **Database:** Prisma / PostgreSQL (Configurable)

## 🚀 যেভাবে শুরু করবেন

### Prerequisites
আপনার কম্পিউটারে **Node.js** ইন্সটল করা থাকতে হবে।

### ধাপসমূহ

১. **রেপো ক্লোন করুন:**
   ```bash
   git clone https://github.com/nullcove/ledger-knox.git
   cd ledger-knox
   ```

২. **ডিপেন্ডেন্সি ইন্সটল করুন:**
   ```bash
   npm install
   ```

৩. **এনভায়রনমেন্ট সেটআপ:**
   `.env.example` ফাইলটি কপি করে `.env.local` তৈরি করুন এবং প্রয়োজনীয় ডাটা দিন।

৪. **রান করুন:**
   ```bash
   npm run dev
   ```
   এখন ব্রাউজারে `http://localhost:3000` ভিজিট করুন।
